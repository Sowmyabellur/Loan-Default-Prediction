# Loan Default Prediction

## Objective
To predict whether a loan will be approved or rejected based on applicant details.

## Dataset
Loan Prediction dataset from Analytics Vidhya.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Random Forest Classifier

## Steps Performed
- Data cleaning and preprocessing
- Label encoding of categorical variables
- Train-test split
- Model training using Random Forest
- Model evaluation using accuracy, precision, recall, and F1-score

## Results
The model achieved good accuracy and was able to correctly identify loan approval patterns.

## Conclusion
This project demonstrates a basic classification workflow suitable for real-world financial prediction problems.
